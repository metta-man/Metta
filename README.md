# Metta

Daily Metta (Loving-Kindness) Meditation App

## Build

```bash
cd metta
xcodegen generate
open Metta.xcodeproj
```

Then build in Xcode (⌘B).

## Features

- Daily metta meditation timer
- Streak tracking
- Session history
- Dark mode UI

## Design System

Uses Metta Labs design tokens:
- Primary: Teal (#2DD4BF)
- Background: Dark (#121214)
